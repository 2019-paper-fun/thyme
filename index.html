Thyme by joaomoreno

A timer for OS X. As simple as that.

  • Open in GitHub
  • Download v0.4.1

Time Your Tasks

Use Thyme to control how much time you spend on your tasks.

Keep a Record

Thyme remembers the time you've spent when using it, so you don't have to.

Global Keyboard Shortcuts

You can use global keyboard shortcuts to trigger Thyme.

New in v0.4 You can now customize the keyboard shortcuts using the new Preferences dialog.

Actionscript

Thyme supports four different Actionscript commands: start, stop, pause and toggle. Here's how:

tell application "Thyme"
    start
end tell
