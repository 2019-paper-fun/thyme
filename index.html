Thyme is a timer for OS X.
It's as simple as that.

With Thyme you can keep track of the time you spend doing your tasks.

Thyme also remembers every time you use it, just so that you don't have to worry about it anymore.

Easter Egg

Ctrl-T is a global hotkey that starts and pauses Thyme. Try it!

Download & Install

Once you get the latest binary, you can drag Thyme to your Applications folder.

Source Code

You clone the project with git by running: 

$ git clone git://github.com/joaomoreno/thyme

get the source code on GitHub : joaomoreno/thyme
