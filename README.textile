h2. Introduction

Thyme is a standalone application which times your tasks, one by one.

h2. Usage

Open the application and notice that now you have a timer on your OS X status bar. You can click it to start or stop it. That's it!

h2. Shortcut

The start / stop action can be triggered by the keyboard shortcut: Ctrl+T.
